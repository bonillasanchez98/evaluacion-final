# Sistema de Gestion Matricula Academica
Este sistema consiste en desarrollar una aplicación backend en Java/Spring Boot, donde se va a llevar a cabo el control de registros de matrículas de estudiantes de una academia, donde se almacenarán en una base de datos los datos de los estudiantes, los cursos y los cursos en que los estudiantes estén matriculados.

## Tecnologías | Herramientas
- Java | Spring
  - Spring Data JPA
  - Lombok
  - Model Mapper
  - Validation
- Postgres
- Postman
